# ⚡ Antigravity YT Downloader

แอปพลิเคชัน Desktop GUI สำหรับดาวน์โหลดวิดีโอจาก YouTube ด้วยภาษา Python และ CustomTkinter ที่พัฒนาขึ้นเป็นพิเศษเพื่อเน้นความเร็วสูงสุด ปลอดภัย และสามารถเลือกความละเอียดที่หลากหลาย (รวมทั้งการดาวน์โหลดเฉพาะไฟล์เสียง MP3 ความละเอียดสูง)

---

## ✨ ฟีเจอร์หลัก (Key Features)

* **📥 ดาวน์โหลดวิดีโอเดี่ยว (Single Download):** ใส่ลิงก์เดี่ยว ดึงข้อมูลปกภาพ หน้าต่างวิดีโอ ช่อง รายละเอียดความยาว และเลือกความละเอียดได้ทันที (ตั้งแต่ 360p, 480p, 720p, 1080p Full HD, 2K, ไปจนถึง 4K)
* **📋 ดาวน์โหลดเพลย์ลิสต์ (Playlist Download):** ดึงรายชื่อคลิปทั้งหมดในเพลย์ลิสต์มาแสดงผล พร้อมให้เลือกเครื่องหมายถูกเฉพาะคลิปที่ต้องการโหลด และเลือกความละเอียดพร้อมกันได้ทุกคลิปในปุ่มเดียว
* **🎵 ดาวน์โหลดเฉพาะเสียง (Audio Extraction):** แปลงไฟล์คลิปเสียงเป็น MP3 คุณภาพสูง 192kbps โดยอัตโนมัติ
* **⏳ คิวดาวน์โหลดอัจฉริยะ (Download Queue):** จัดการคิวดาวน์โหลดแบบเป็นลำดับ เพื่อรักษาเสถียรภาพแบนด์วิธและการทำงานของเครื่อง โดยที่หน้าต่าง GUI จะยังคงลื่นไหลและตอบสนองได้ตลอดเวลา (ไม่ค้าง)
* **⚙️ ติดตั้ง FFmpeg อัตโนมัติ (Portable Auto-Installer):** หากโปรแกรมตรวจไม่พบ FFmpeg ในระบบ จะทำการดาวน์โหลดและติดตั้งตัวเสริมนี้ให้ทันทีแบบเงียบๆ ทำให้คุณโหลดวิดีโอความละเอียด 1080p+ หรือแยกไฟล์เสียงได้ทันทีแบบแกะกล่อง
* **📜 ระบบประวัติการดาวน์โหลดสำเร็จ (Download History):** บันทึกประวัติการดาวน์โหลดสำเร็จลงในฐานข้อมูล SQLite ภายในเครื่อง สามารถกดคลิกเปิดโฟลเดอร์ไฟล์หรือลบรายการได้ง่ายดาย
* **🎨 ปรับแต่งธีมสุดหรู (Settings):** มาพร้อม Dark Mode ที่เป็นมิตรต่อสายตา หรือเลือกเปลี่ยนธีมตามความต้องการ

## 🛠️ วิธีการติดตั้งและรันใช้งาน (Installation & Launch)

### สำหรับระบบปฏิบัติการ Windows (ผ่าน PowerShell หรือ Command Prompt):

1. **โคลนหรือสร้างโฟลเดอร์โปรเจกต์** และเปิด Terminal ในโฟลเดอร์ `yt-download/`
2. **สร้าง Virtual Environment ในโฟลเดอร์โปรเจกต์** เพื่อป้องกันการชนกันของไลบรารี:
   ```powershell
   python -m venv .venv
   ```
3. **ติดตั้งไลบรารีทั้งหมดที่จำเป็น** ผ่านไฟล์ `requirements.txt`:
   ```powershell
   .venv\Scripts\pip.exe install -r requirements.txt
   ```
4. **เปิดใช้งานแอปพลิเคชัน**:
   ```powershell
   .venv\Scripts\python.exe main.py
   ```

## 📁 โครงสร้างโปรเจกต์ (Project Structure)

```text
yt-download/
│
├── .venv/                  # Virtual Environment สำหรับเก็บไลบรารีเฉพาะของโปรเจกต์
├── bin/                    # เก็บโปรแกรมพกพา FFmpeg (ระบบดาวน์โหลดให้คุณโดยอัตโนมัติ)
├── data/                   # เก็บฐานข้อมูล SQLite สำหรับแสดงหน้าประวัติการดาวน์โหลด
│
├── downloader.py           # ตัวขับเคลื่อนควบคุมเอนจิ้นหลัก (yt-dlp, FFmpeg installation, Format parser)
├── queue_manager.py        # ระบบจัดคิวดาวน์โหลดเบื้องหลังแบบ Multi-threaded
├── history_db.py           # คลาสจัดการฐานข้อมูลประวัติการดาวน์โหลดสำเร็จผ่าน SQLite
├── main.py                 # หน้าอินเตอร์เฟสและ UI หน้าต่างควบคุมด้วย CustomTkinter
│
├── requirements.txt        # ไลบรารี Python ที่แอปพลิเคชันนี้ใช้
└── README.md               # คู่มือชี้แจงการติดตั้งและการทำงานฉบับนี้
```
